# DevMind AI IDE

DevMind is a premium, browser-based AI Integrated Development Environment (IDE) designed for modern software engineering workflows. It combines powerful code editing with advanced AI orchestration, allowing developers to build, edit, and analyze codebases with unparalleled speed.

## üöÄ Key Features

- **Agentic Coding Assistant**: Describe your changes in natural language, and DevMind will generate precise file modifications using a robust XML patching protocol.
- **Visual Context Analysis**: Upload images or PDFs to provide the AI with visual context for UI/UX development or document-based coding tasks.
- **Multi-Provider Support**: Seamlessly switch between Google Gemini, OpenAI, Anthropic, OpenRouter, and local LLM endpoints.
- **Thought-Process Visualization**: Watch the AI's reasoning in real-time with an expandable "Thought Process" block.
- **Smart Explorer**: Full drag-and-drop file management, folder structures, and workspace export capabilities.
- **Themeable UI**: Professional-grade themes including Dark, Light, Midnight Blue, and Solarized Dark.
